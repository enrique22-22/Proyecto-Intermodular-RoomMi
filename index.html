RoomMi

Gestión de alquiler de habitaciones · 1º DAM

Plataforma diseñada para conectar a propietarios con inquilinos de forma eficiente y segura. El proyecto integra múltiples módulos del ciclo formativo y cuenta con una base de datos relacional normalizada hasta 3FN.

🗄️ MySQL

👤
Gestión de usuarios

🏠
Publicación de habitaciones
